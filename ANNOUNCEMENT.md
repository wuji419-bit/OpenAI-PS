# OpenAI Photoshop Generator v0.1.70 公告

本次更新主要修复 Photoshop 面板里的抠图位置、按钮显示和设置页排版问题，并补充抠抠图官网入口。
新增 `拆图` 模式：使用 `gpt-image-2` 先识别画面里的独立元素，再把返回的透明 PNG 自动拆成多个 Photoshop 图层。
修复透明结果预览过小的问题：预览会自动裁到可见像素范围，但导入 Photoshop 的原始尺寸和位置不变。
继续放大结果区：缩略图改为单列大图，上方选中预览也加高，方便直接检查拆图/抠图细节。
拆图逻辑改为语义拆图：`gpt-image-2` 会分别抽取底框/外框、填充条、徽章、文字数字等同尺寸透明图层，不再只按连通区域合并成一个整体。
拆图现在会先自动识别画面里应该拆开的元素，再逐层调用 `gpt-image-2` 输出透明图层；正常情况下不需要手写“血条底框、红色条、骨头牌、x10 字”这类清单。
修复自动拆图标签丢失问题：放回 Photoshop 的图层会保留识别名称，例如 `Split 1 血条底框`、`Split 2 红色条`、`Split 3 骨头牌`、`Split 4 x10 字`。

## 更新内容

- 修复抠抠图结果自动放回 Photoshop 后发生偏移的问题。
- 优化 Photoshop UXP 面板布局，修复模式按钮、设置页按钮、参数控件显示异常。
- 去掉没有实际操作意义的模式说明卡片，减少界面占用。
- 修复扩图边距控件位置异常，改为更稳定的左对齐输入布局。
- 在抠抠图 API 设置区增加官网和 API 文档入口：
  - 官网：https://www.koukoutu.com/
  - API 文档：https://www.koukoutu.com/dev
- 同步源插件目录和 Photoshop 运行目录，避免两边版本不一致导致按钮失效。
- 新增拆图：当前画布会走 OpenAI 图片编辑接口和 `gpt-image-2`，插件再按透明元素自动分层放回 Photoshop。
- 优化结果预览：抠图和拆图这类透明 PNG 会用放大的裁切预览显示，方便检查细节。
- 结果缩略图改成单列大图，避免两列小格子看不清。
- 拆图模式改为多次语义抽取，适合血条、按钮、徽章、文字这类互相接触但需要拆成独立图层的 UI 元素。
- 拆图模式新增自动元素识别：会先分析当前画面，再按识别出的元素列表分层，手写拆分目标只作为高级兜底用法。
- 修复自动识别出的拆图标签没有传到最终 Photoshop 图层名的问题。

## 使用提示

更新后请关闭插件面板并重新打开；如果仍看到旧界面，请重启 Photoshop。新版界面顶部显示版本号 `v0.1.70`。

抠图模式仍使用当前配置的抠抠图 API Key，不会改变已有密钥和接口设置。
