# OpenAI Photoshop Generator v0.1.152 公告

这次更新主要修正 `v0.1.151` 在本机 Cockpit Tools 端口上的误判。

## 更新内容

- 默认 Base URL 改回当前可用的 `http://127.0.0.1:49456/v1`。
- 不再把 `49456` 自动改成当前未运行的 `51866`。
- `49456` 会被识别为 Cockpit/Codex 本地 sidecar，选区重绘会优先请求 `/images/edits`，失败时才回退 `/responses`。
- 每次生成前会清空“当前结果”列表，旧图只留在“历史”里，避免数量 1 却显示多个旧缩略图。
- 针对中文“弓”增加提示词消歧：弓指武器弓箭，不是蝴蝶结、领结、缎带或装饰结。

## 使用提示

更新后请关闭并重新打开 Photoshop 插件面板；如果顶部版本不是 `v0.1.152`，请重启 Photoshop。
