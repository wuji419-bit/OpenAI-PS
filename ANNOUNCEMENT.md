# OpenAI Photoshop Generator v0.1.73 公告

这次更新重点修正 `拆图` 的真实工作流：不再直接相信 `gpt-image-2` 的透明输出，而是先让它把每个语义图层画在纯色底上，再交给抠抠图移除背景，最后按原画布尺寸和原坐标导回 Photoshop。

## 更新内容

- `拆图` 增加 `拆图背景` 选项：`自动`、`黑底`、`白底`。
- 默认 `自动` 会根据目标层内容选择黑底或白底；边缘不好时可以手动强制黑底/白底重新跑。
- `gpt-image-2` 每次只抽取一个语义元素，并保持整张画布尺寸，不裁切、不居中、不移动。
- 抽取出来的纯色底图会继续调用抠抠图，生成透明 PNG，再导入 Photoshop 分层。
- `拆图` 现在也要求填写抠抠图 API Key，因为透明背景清理交给抠抠图完成。
- 修复本地 `http://127.0.0.1:49456/v1` 中转返回 `/responses` SSE 流时，插件解析失败导致自动识别不稳定的问题。
- 保留 v0.1.72 的定位逻辑：拆出来的层仍按原始画布大小和原始坐标叠回去，只是在 PS 里面分层。

## 使用提示

更新后请关闭插件面板并重新打开；如果仍看到旧界面，请重启 Photoshop。新版界面顶部显示版本号 `v0.1.73`。

如果元素边缘偏亮、偏透明，优先试 `黑底`；如果元素本身很暗或文字是黑色，试 `白底`。
