# Maintainer Plan

OpenAI Photoshop Generator is maintained as a practical open-source workflow project for Photoshop users who want OpenAI image generation and editing inside their creative process.

## Maintainer Responsibilities

- Track OpenAI Image API changes and update the plugin defaults.
- Keep Photoshop UXP compatibility notes current.
- Maintain generation, reference edit, selection repaint, and outpaint flows.
- Review issues for reproducible bugs and workflow requests.
- Publish releases with manual smoke-test notes.
- Keep user API keys and creative assets out of the repository.

## Codex and API Credit Use

If this project receives API credits, they will be used for:

- regression testing generation and edit workflows
- verifying model, size, quality, and format compatibility
- testing reference-image, inpaint, and outpaint requests
- creating repeatable demo assets for documentation
- testing release candidates against real Photoshop workflows
- maintaining issue triage, changelog, and release automation

Credits will not be used for unrelated commercial image production.

## Evidence to Keep Updated

- README screenshots or demo links
- release notes
- issue discussions with reproducible workflows
- compatibility notes for Photoshop versions
- API migration notes when OpenAI image models or parameters change
