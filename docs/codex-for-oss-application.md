## Why the Repository Qualifies

OpenAI PS is a public MIT-licensed Photoshop UXP plugin that integrates OpenAI image generation and editing into a real creative desktop workflow. It supports text-to-image, current-canvas reference edits, rectangular-selection repainting, outpainting, result preview, history, and importing generated results back into Photoshop layers. The project was developed with Codex assistance and is maintained as an open workflow for Photoshop users and plugin developers.

## API Credit Use

API credits would be used to test OpenAI Image API generation and editing workflows, verify Photoshop plugin releases, reproduce user issues, produce demo assets, and maintain compatibility as GPT Image models and parameters evolve.
